# 파이썬을 활용한 리눅스 파일 생성 실습

본 문서는 VS Code 및 리눅스 터미널 환경에서 파이썬 프로그램을 작성하고, 루트 디렉토리 하위에 파일을 안전하게 생성 및 확인하는 과정을 정리한 가이드입니다.

---

## 1. 실습 환경 및 파일 위치
* **작업 디렉토리:** `~/study/linux`
* **파이썬 프로그램 파일명:** `task2_pb1.py`
* **목표 생성 파일:** `/test/hello.txt` (내용: `Hello Linux`)

---

## 2. 프로그램 예외 없는 실행 방법 (핵심 해결책)

리눅스에서 루트 디렉토리(`/`) 바로 아래에 임의로 `/test` 디렉토리를 만들거나 파일을 쓰려고 하면 시스템 보안상 **`PermissionError` (권한 거부)** 가 발생합니다. 프로그램이 예외 없이 완전히 동작하도록 만들기 위해 아래의 방법 중 하나를 선택해야 합니다.

### [해결책] 관리자(Root) 권한으로 프로그램 실행
스크립트 내부에서 예외 처리를 구현함과 동시에, 터미널에서 실행할 때 `sudo` 명령어를 사용하여 파일 시스템 쓰기 권한을 확보합니다.

```bash
# 디렉토리 이동
cd ~/study/linux

# sudo 권한으로 파이썬 프로그램 실행
sudo python3 1_hello.py
